<p><strong>Роль: </strong>участие в разработке программного обеспечения для оказания дополнительных услуг абонентам мобильной связи на основе SMS, MMS, USSD, а также других транспортов и технологий. Разработка бизнес-логики, пользовательских web-интерфейсов с использованием JSP и сервлетов, реализация интеграционных модулей (HTTP, XML/SOAP, сокетные протоколы).</p> <p> </p> <p><strong>Должностные обязанности:</strong></p> <ul> <li>программирование (разработка кода на java / JSP / SQL, реализующего поставленные задачи)</li> <li>участие в проектировании архитектуры системы</li> <li>участие в согласовании требований (анализ поставленной задачи, оценка выполнимости, формулировка уточняющих вопросов, оценка трудоемкости)</li> <li>участие в документировании</li> <li>участие в разборе инцидентов, на третьем уровне поддержки, поиск и исправление ошибок в коде</li> </ul> <p> </p> <p><strong>Обязательные требования:</strong></p> <ul> <li>знание языка java</li> <li>знание и понимание web-технологий, знание HTML/CSS, JavaScript, Ajax, JQuery и т.п.</li> <li>понимание реляционных БД, знание SQL</li> <li>желание работать в команде в роли разработчика программного обеспечения и совершенствоваться в этом направлении</li> </ul> <p> </p> <p><strong>Дополнительные требования (любой из перечисленных пунктов будет дополнительным плюсом для кандидата на вакансию)</strong></p> <ul> <li>опыт работы с PostgreSQL (предпочтительно) либо Oracle, опыт программирования логики на сервере СУБД</li> <li>понимание механизмов работы многопоточных систем в целом и реализованных на java в частности</li> <li>опыт разработки сложных web-интерфейсов на JSP, в том числе с использованием JQuery / Bootsrap / Google Charts и т.п.</li> </ul> <p> </p> <p><strong>Условия:</strong></p> <ul> <li>испытательный срок - 2 месяца</li> <li>работа в офисе</li> </ul>
