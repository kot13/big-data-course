<p>Мы разрабатываем технологические решения, которые помогают крупнейшим участникам рекламного рынка размещать рекламу в социальных сетях.</p> <p><br /><strong>Наши плюсы:</strong></p> <ul> <li>молодой коллектив, почти полностью состоящий из выпускников технических вузов</li> <li>просторный офис в центре Москвы (метро улица 1905 года, Краснопресненская, Баррикадная)</li> <li>начало рабочего дня в 11:00.</li> <li>в работе используем aws, ec2, php, zf2, doctrine 2, jquery, redis, mysql, nginx, svn.</li> <li>мы ищем того, кто поможет нам развивать систему для автоматизированного управления таргетированной рекламой в соцсетях (Таргет@Mail.ru, ВКонтакте, Facebook). Перед вами будут стоять нетривиальные задачи работы с рекламными API и значительными объемами данных. В процессе работы вы сможете близко познакомиться с размещением, управлением, оценкой эффективности рекламных кампаний для крупнейших брендов.</li> </ul> <p> </p> <p><strong>Обязательно:</strong></p> <ul> <li>высокий уровень знаний в php</li> <li>иметь опыт работы с ZF2 (Yii, Symfony) от 1 года</li> <li>высокий уровень знаний mysql</li> <li>базовые знания js, jquery</li> </ul> <p> </p> <p><strong>Большим плюсом будет:</strong></p> <ul> <li>опыт работы с API соцсетей (vk, fb, mail.ru)</li> <li>понимание основ интернет-рекламы (что такое cpm, cpc, cpa, ctr, конверсия)</li> <li>опыт работы с redis</li> <li>опыт работы с AngularJS, Backbone.js.</li> </ul>
