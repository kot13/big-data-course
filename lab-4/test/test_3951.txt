<p><strong>Описание проекта:</strong></p> <p>Исходные данные: Имеется библиотека на C++, которая позволяет выполнять основные телефонные функции – производить звонок по заданному номеру телефона, принимать входящий звонок, вызывая на него некий callback. В библиотеке реализованы некоторые дополнительные сервисные функции. Над ней написана обертка на managed C++, которая вызывается из C# WPF приложения.<br /><br /><strong>Задача: </strong></p> <ul> <li>Реализовать для начала технологический прототип приложения, которое реализует функции данной библиотеки или использует эту библиотеку и может вызвать браузер IE с определенным URL-ом и данными входящего звонка (на какой номер позвонили, с какого номера позвонили и дополнительная cti-информация).</li> <li>Необходима возможность набора номера телефона со страницы веб-приложения, например, кликом по ссылке с номером телефона.<br /></li> </ul> <strong>Обязательные требования:</strong> <ul> <li>Компетенция С++, чтобы разобраться в существующей библиотеке, написанной на C++.</li> <li>Компетенция в использовании API Avaya и опыт или знание SDK Avaya, которое интенсивно используется в текущей библиотеке.</li> <li>Компетенция в C#, чтобы реализовать описанную задачу, т.к. разработку ПО мы ведем на C#.</li> </ul> <p> </p> <p><strong>Условия:</strong></p> <ul> <li>Участие в проектах Компании Федерального уровня;</li> <li>Строгое соблюдение ТК РФ;</li> <li>Офис - м.Кожуховская (шаговая доступность).</li> </ul> <p> </p>
