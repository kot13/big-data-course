<p> </p> <p><strong>Клиент – американская</strong> фирма с отделом в Люблине. Фирма занимается разработкой приложений в отрасли медицины и фармакологии.</p> <p> </p> <p><strong>Задания:</strong> работа в роли java developer, работа в команде, которая разрабатывает медицинское программное обеспечение класса HIS на платформе Java.</p> <p><strong>Кроме того:</strong></p> <ul> <li>создание автоматических тестов ( Java + JavaScript )</li> <li>забота о качестве написанного кода ( тестов )</li> <li> документация ( sphinx/confluence )</li> </ul> <p> </p> <p><strong>Требования</strong><strong>:</strong></p> <ul> <li>java + ORM ( Hibernate ) + DI ( Spring )</li> <li>знание AngularJS, IntelliJ IDEA будет плюсом</li> </ul> <p> </p> <p><strong>В социальный пакет входит:</strong></p> <ul> <li>Командировки по Европе и в Америку</li> <li>Медстраховка</li> <li>Языковые курсы (немецкий, английский, русский, польский)</li> <li>Абонементы ( тренажерный зал, фитнес )</li> <li>Групповые выезды с фирмой на отдых</li> <li>Международные конференции</li> </ul>
