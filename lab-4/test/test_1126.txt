<p>Сервис «Мобильный консультант» приглашает практикующих специалистов различных сфер разработки ПО для проведения экспертного консалтинга. Требуются ИТ-консультанты для информационной поддержки разработчиков в решении технических задач. Создание сервиса «Мобильный консультант» осуществляется в содействии с ведущими операторами региона: МТС, Мегафон, Мотив.</p> <p> </p> <p><strong>Обязанности:</strong></p> <ul> <li>Консультирование пользователей по вопросам сопровождения и доработки ИС;</li> <li>Советы по разработке WEB-приложений на Javascript и других языках;</li> <li>Инструктирование по доработке типовых решений под задачи компании;</li> <li>Консультирование пользователей по вопросам сопровождения и доработки ИС;</li> <li>Советы по разработке WEB-приложений на Javascript и других языках.</li> </ul> <p> </p> <p><strong>Требования:</strong></p> <ul> <li>Глубокое знание T-SQL;</li> <li>Опыт разработки WEB-приложений на Javascript;</li> <li>Знание CSS, HTML;</li> <li>Опыт написания софта на продажу;</li> <li>Наличие завершенных проектов;</li> <li>Хорошее владение платформой Android;</li> <li>Знание типовых решений 1С;</li> </ul> <p> </p> <p><strong>Условия работы:</strong></p> <ul> <li>Совместительство;</li> <li>Гибкий график.</li> </ul>
