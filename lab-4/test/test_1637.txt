<p>Необходимые знания: <strong>ASP.NET 4.x</strong> (лучше — <strong>4.5</strong>), шаблоны проектирования, базовые алгоритмы, умение самостоятельно мыслить, знание английского языка (желательно intermediate и выше).</p> <p>Текущий проект написан на <strong>ASP.NET</strong> <strong>MVC 4.0</strong>. Текущая инфраструктура состоит из <strong>Microsoft SQL Server (T-SQL)</strong> и <strong>ASP.NET / MVC</strong>. Используется <strong>MS IIS</strong>.</p> <p>Обязанности: Поддержка, последующее сопровождение и миграция существующего проекта для лидера в сфере образования США. Стек технологий с которыми прийдется иметь дело: <strong>Microsoft SQL Server 2012 R4, ASP.NET MVC 4.0, MSMQ, REST Services</strong> (знание не обязательно, научим), <strong>Microsoft Team Foundation Server, Git, Jira</strong>.</p> <p>Условия работы: Работа в небольшой команде до четырех человевек по Agile-процессам в тесном сотрудничестве с иностранными заказчиками. Стабильная продуктовая компания, на рынке более 10 лет. Просторный офис близко от центра. Большая дружная команда, нас уже 100+ человек. Мы готовы рассмотреть кандидатов, которые могут сменить технологию с Java или имеют опыт работы с ней. Позитивный опыт смены парадигмы у нас имеется.</p> <p>Предлагаем:</p> <ul> <li>Достойная заработная плата.</li> <li>Гибкий график работы.</li> <li>Зарубежные командировки.</li> <li>Оплачиваемые переработки, отпуск и больничные.</li> <li>Обучение иностранному языку.</li> <li>Корпоративные праздники и встречи.</li> <li>Отсутствие дресс-кода.Зона отдыха.</li> <li>Аренда спортивного зала.</li> <li>Офис в комфортабельном БЦ.</li> <li>Фруктовые завтраки и ужины.</li> <li>Вкусный чай и кофе в неограниченных количествах.</li> </ul>
