<p>Код вакансии: Java программист (разработчик, developer)</p> <p>Axibase специализируется на разработке программных продуктов для оптимизации ИТ инфраструктуры. В 2013 году компания выпустила принципиально новое решение для хранения и анализа временных рядов с использованием нереляционных баз данных - Axibase Time-Series Database. Приглашаем на работу back-end Java специалистов.</p> <p>Ознакомиться с ATSD можно на http://axibase.com/axibase-time-series-database/</p> <p><strong>Области разработки: </strong></p> <ul> <li>back-end/persistence</li> <li>back-end/data collection</li> <li>back-end/messaging</li> <li>REST API</li> <li>Query DSL: custom SQL dialect, custom JDBC driver</li> <li>Network protocols: TCP, UDP, MQTT, COAP</li> </ul> <p> </p> <p><strong>Технологический stack: </strong></p> <ul> <li>Java 1.6/1.7</li> <li>HDFS/HBase</li> <li>Netty</li> <li>Jetty</li> <li>Kafka</li> <li>Spring</li> </ul> <p> </p> <p><strong>Требования:</strong></p> <ul> <li>Опыт программирования на Java от 5-ти лет.</li> <li>Участие в разработке решений, продаваемых по лицензии, т.е. не заказная разработка и не разработка для внутренних служб.</li> <li>Отличное значение Spring, включая core, integration, batch, MVC.</li> </ul> <p> </p> <p><strong>Пожелания:</strong></p> <ul> <li>Письменный английский на уровне 4+, так как внутреннее управление задачами ведется на англ. языке.</li> </ul> <p> </p> <p><strong>Наш офис:</strong></p> <ul> <li>Офис класса А в центре города.</li> <li>Рядом с метро Маяковская / Площадь Восстания, 3 минуты от Невского проспекта.</li> <li>Бесплатная парковка для велосипедистов.</li> </ul> <p> </p> <p><strong>Преимущества:</strong></p> <ul> <li>Стабильность, основанная на производстве собственных продуктов для клиентов класса enterprise.</li> <li>Интересный продукт, современный стэк.</li> <li>Удобное расположение офиса в самом центре Санкт-Петербурга.</li> </ul>
