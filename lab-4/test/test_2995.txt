<p>Компания love2shop, стартап в сфере fashion, приглашает амбициозных специалистов в свою команду.</p> <p><strong>Обязанности:</strong></p> <ul> <li>Разработка RIA web-app, API сервиса и административных инструментов</li> </ul> <p><br /><strong>Требования:</strong></p> <ul> <li>Хорошее знание PHP5 (&gt;=5.5), MySQL (включая оптимизацию запросов), <em>javascript</em>.</li> <li><em>Angular</em>.<em>js</em>, <em>Symfony2</em>, <em>Doctrine2</em>, <em>PHPUnit</em>, <em>composer</em>, <em>gulp</em>/<em>grunt</em>, <em>bower</em>, <em>git</em>.</li> <li>Знание design patterns, стремление соблюдать принципы KISS и SOLID.</li> <li>Знать как устроен HTTP-протокол, что такое REST, для чего нужен CORS.</li> <li>Опыт работы с HTML5 API - websocket, webstorage, canvas, drag-n-drop, history.</li> <li>Умение верстать (pixel-perfect, адаптив), CSS3, SASS(SCSS).</li> <li>Понимание специфики работы с высокими нагрузками, умение найти узкое место в приложении (с помощью профайлера и головы).</li> <li>Опыт построения RESTfull-приложений.</li> </ul> <p><br /><strong>Условия:</strong></p> <ul> <li>Интересные задачи и неограниченные перспективы развития;</li> <li>Работа в профессиональной творческой команде;</li> <li>Гибкий график;</li> <li>Достойная заработная плата + премии + опционы;</li> <li>Образовательные программы;</li> <li>Офис в пространстве Ткачи (3 мин от м. Обводный канал);</li> </ul>
