<p>Профессиональная команда приглашает Разработчика хранилищ данных для разработки и сопровождения программного продукта Neoflex Reporting.</p> <p>Продукт представляет собой хранилище данных с возможностью формирования большого пула аналитических отчетов для пользователя, в первую очередь форм обязательной отчетности для Центрального банка.</p> <p>Хранилище продукта строится на данных из других систем банка при помощи ETL - средств.</p> <p>В направлении решаются задачи, как разработки нового функционала для данного продукта, так и внедрения всех его компонентов (от загрузки данных до формирования отчетности) у новых клиентов.</p> <p> </p> <p><strong>Обязанности:</strong></p> <ul> <li>Разработка и сопровождение отчетных форм Центрального банка в продукте Neoflex Reporting для отдельных клиентов и в продукте в целом;</li> <li>Написание кода на PL\SQL;</li> <li>Разработка отчетов на Cognos BI;</li> <li>Разработка отчетов на SAP BO;</li> <li>Разработка приложений на Oracle APEX;</li> <li>Разработка маппингов загрузки данных с помощью ETL – инструментов.</li> </ul> <p> </p> <p><strong>Необходимые профессиональные навыки и знания:</strong></p> <ul> <li>Опыт проектирования баз данных (желательно для банковской предметной области);</li> <li>Опыт работы (понимание архитектуры) с СУБД Oracle от 2-х лет;</li> <li>Опыт работы с Oracle APEX;</li> <li>Навыки работы с PL\SQL Developer, SQL Developer (или их аналог);</li> <li>Навыки построения плоских отчетов и olap кубов;</li> <li>Навыки работы с системами контроля версий (svn);</li> <li>Знание Python, Java приветствуется;</li> <li>Знание основ банковского бухгалтерского учета;</li> <li>Умение разрабатывать шаблоны отчетов с помощью инструментов BI;</li> <li>Умение работать в команде, несмотря на территориальную разрозненность отдела;</li> <li>Желание разбираться в задачах до мелких деталей.</li> </ul> <p><strong>Владение программными продуктами:</strong></p> <ul> <li>СУБД Oracle 10g/11g/12с и Mircosoft SQL Server 2003-2008;</li> <li>Informatica Power Center или IBM Data Stage или Talend;</li> <li>SAP BO и/или IBM Cognos.</li> </ul> <p><br /><strong>Условия:</strong></p> <ul> <li>Трудоустройство согласно ТК РФ;</li> <li>График работы 5/2; режим работы с 9:00 до 18:00</li> <li>Добровольное медицинское страхование, включая стоматологию;</li> <li>Внутреннее и внешнее обучение и сертификация.</li> </ul>
