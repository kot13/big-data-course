<p>Американская корпорация Align Technology, Inc., занимающаяся исследованиями, разработкой и производством в области цифровой медицины, приглашает на работу в московском R&amp;D центре разработчика Java.</p> <p> </p> <p><strong>Что </strong> <strong>делает команда:</strong></p> <ul> <li>Основная задача команды – создание консистентного API между разными системами для предоставления доступа и управления клиническими данными как для внутренних систем компании, так и для внешних пользователей. Данный API можно охарактеризовать скорее как набор приложений, взаимодействующих между собой, нежели как одно монолитное приложение</li> <li>В дополнении к API команда будет привлекаться для задач связанных с клиническими данными в новых проектах компании.</li> <li>Кроме того команда будет заниматься интеграционными задачами.</li> </ul> <p> </p> <p><strong>Как делаем?</strong></p> <ul> <li>API представляет собой набор веб приложений. Они отличаются по задачам, но общие черты следующие:</li> <li>Хостинг в облаке (Amazon AWS)</li> <li>Сервер – Tomcat 7, java 7. Осенью планируется переехать на 8 версии.</li> <li><strong>Frameworks:</strong> Spring (MVC, Data, Security)</li> <li><strong>Database</strong>: из реляционных будет использоваться PostgreSQL, из NoSQL решения от Amazon (DynamoDB)</li> <li><strong>Infrastructure:</strong> Git, Bamboo, Chef, Jira, Confluence, Fisheye</li> </ul> <p><strong>От программиста </strong> <strong>нужно:</strong></p> <ul> <li>Опыт разработки на Java в коммерческих проектах от 2 лет</li> <li>Опыт разработки веб приложений от 2 лет</li> <li>Уверенные знания платформы JavaSE</li> <li>Опыт использования Spring Framework (MVC, Data, Security, messaging)</li> <li>Хорошее знание протокола HTTP</li> <li>Знание концепций баз данных SQL</li> </ul> <p> </p> <p><strong>Здорово если Вы:</strong></p> <ul> <li>Знаете что такое REST, HATEAOS и у вас был опыт создания Web API или хорошие теоретические знания по вопросу</li> <li>Знаете, зачем нужны гипермедиа типы общего назначения и почему application/json к ним не относится</li> <li>Имели опыт работы с Amazon AWS</li> </ul> <p> </p> <p><strong>Что делать </strong> <strong>в проекте?</strong></p> <ul> <li>Основной задачей является разработка Web API для клинической части данных, полностью удовлетворяющим REST подходу.
